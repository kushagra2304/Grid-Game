# Multiplayer Unicode Grid

A real-time multiplayer web app where users can place Unicode characters on a shared 10x10 grid.

## Features
- Shared grid updated in real-time
- Displays online player count
- Once a player updates, they canâ€™t update again
- Built with React + Node.js (TypeScript + Socket.IO)

## Tech Stack
- Frontend: React (Vite + TypeScript) + TailwindCSS
- Backend: Node.js, Express, TypeScript, Socket.IO

## Setup Instructions

### Start the Backend
cd backend
npm install
npm run dev

### Start the Frontend
cd client
npm install
npm run dev
